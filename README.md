## `creative-ui`

## Get started

Install

```bash
yarn add creative-ui
# or
npm install --save creative-ui
```

Use

```typescript
import { Loader } from 'creative-ui'

const App = () => {
  return (
    <div>
      <Loader progress={0.5}></Loader>
    </div>
  )
}
```

[Examples](https://github.com/skulptur/creative-ui/tree/master/example)
